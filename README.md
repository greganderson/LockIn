# LockIn

This is a tool to help make it easier to read long boring documents (I'm looking at you, [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)). It was born of a "man, there's got to be a way AI can help make this easier to get through." Sure enough, it can!

All you need to run this is to open the `index.html` and drag the LockIn button onto your bookmarks bar. Then you can just click the button while on any site and it'll pull up the controls.

## ~Recipe Backstory~ Why I Made This

I was browsing through a site one of my students made (https://adhd.doodleink.net/), which was a purely vibe coded fun site that satisfies all sorts of ADHD itches. It was a lot of fun. Right after that though, I pulled up the [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) to find a section for my students to read before we went over a specific topic. I was running on 4 hours of sleep and I couldn't make it through much of the page before I realized I wasn't actually consuming any of the words I was reading. I thought it might be cool to see if AI could make some kind of tool to help make a page like that more interesting After a couple prompts, I got LockIn.
